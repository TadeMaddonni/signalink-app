# Pasos para TestFlight Gratuito

## 1. Registro Apple Developer (Gratuito)

- Ve a developer.apple.com/register
- Inicia sesión con tu Apple ID
- Acepta términos y condiciones
- NO pagues $99 - solo registro gratuito

## 2. Build para TestFlight

```bash
eas build --platform ios --profile production
```

## 3. Subir a TestFlight

- Usa Application Loader o Xcode
- Sube el .ipa generado
- Invítate como tester interno

## Limitaciones:

- ✅ Bluetooth funciona perfectamente
- ✅ Testing completo en dispositivo real
- ❌ Solo 90 días de validez
- ❌ Máximo 100 testers
- ❌ No puede ir a App Store
